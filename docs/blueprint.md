# **App Name**: ShopWave

## Core Features:

- Product Listings: Display products with images, descriptions, and prices.
- Add to Cart: Allow users to add products to their shopping cart.
- Shopping Cart Management: Enable users to view, modify, and remove items from their cart.
- Theme Toggle: Implement a theme toggle to switch between light and dark modes.
- Pulse Animation: Add a pulse animation to the cart icon when an item is added.

## Style Guidelines:

- Primary color: Light blue (#2EAADD) to provide a calm and inviting shopping experience.
- Background color: Very light blue (#F0F8FF) provides a clean canvas for product display.
- Accent color: Blue (#4682B4), a slightly darker hue, ensures buttons and key elements are noticeable without overpowering the main color scheme.
- Body and headline font: 'Inter', a sans-serif font providing a clean and modern look suitable for e-commerce.
- Responsive layout to ensure compatibility across various devices.
- Use simple, outlined icons for navigation and product categories.
- Subtle pulse animation on the cart icon when items are added.
