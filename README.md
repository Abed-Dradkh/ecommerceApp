# ShopWave - E-commerce Starter

This is a Next.js-based e-commerce application. It's built with modern technologies to provide a solid foundation for your online store.

## Features

- **Modern UI/UX**: A clean, beautiful, and responsive design.
- **Product Catalog**: Easily browse and sort through products.
- **Dynamic Shopping Cart**: Add, remove, and update item quantities seamlessly.
- **Light & Dark Mode**: Switch between themes effortlessly.
- **Performance Optimized**: Built with the Next.js App Router for a fast, server-first experience.
- **Developer-Friendly**: Clean and well-structured code.

Follow these instructions to get a local copy up and running for development and testing purposes.

### Prerequisites

- [Node.js](https://nodejs.org/en/) (v18 or newer)
- npm or yarn

### Installation & Setup

1.  Clone the repository:
    ```sh
    git clone https://github.com/your-username/your-repo-name.git
    ```
2.  Navigate to the project directory:
    ```sh
    cd your-repo-name
    ```
3.  Install dependencies:
    ```sh
    npm install
    ```
4.  Run the development server:
    ```sh
    npm run dev
    ```

The application will be available at `http://localhost:9002`.
